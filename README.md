1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?

Ans : 1. getElementById : it select only a specified unique ID . it is the fast way to select an element.

     2. getElementByClassName : it select all element that have the specified class name. 

     3. querySelector : it select all the valid css selector like id, class, tag etc.
